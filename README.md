# الإسلام يحارب السلبية — الصف الثاني عشر

لعبة ويب تعليمية جاهزة لتدريس **اللغة العربية للصف الثاني عشر**، ومخصصة حاليًا لـ **الوحدة: الدرس الرابع** في مهارات **الفهم والاستيعاب والتحليل**.

## التشغيل

### الطريقة الأسرع

- افتح ملف `index.html` مباشرة في المتصفح.

### أو عبر خادم محلي

من داخل المجلد شغّل:

```powershell
python -m http.server 8000
```

ثم افتح:

```text
http://127.0.0.1:8000
```

## المميزات

- واجهة عربية `RTL`
- أسئلة مخصصة مباشرة لدرس `الإسلام يحارب السلبية`
- تقسيم المسارات إلى: `فهم واستيعاب` و`مفردات ودلالات` و`تحليل`
- مؤقت لكل سؤال
- نقاط فورية وتغذية راجعة
- أفضل النتائج محفوظة محليًا في المتصفح
- مناسبة للمشاركة عبر Teams أو العرض على شاشة الصف

## التخصيص السريع

يمكن تعديل بنك الأسئلة من ملف `script.js` داخل المتغير `questionBank`.

كل سؤال يتكون من:

- `category`
- `prompt`
- `options`
- `correctIndex`
- `explanation`

## الملفات

- `index.html` الواجهة الرئيسية
- `styles.css` التنسيق
- `script.js` منطق اللعبة والأسئلة
