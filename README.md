# الضاد — تعلم بمرح

لعبة ويب تعليمية تفاعلية جاهزة لتدريس **اللغة العربية للصف الثاني عشر**، ومهيأة حاليًا لدرس **الإسلام يحارب السلبية** في مهارات **الفهم والاستيعاب والتحليل**.

## التشغيل

### الطريقة الأسرع

- افتح ملف `index.html` مباشرة في المتصفح.

### أو عبر خادم محلي

من داخل المجلد شغّل:

```powershell
python -m http.server 8000
```

ثم افتح:

```text
http://127.0.0.1:8000
```

## المميزات

- واجهة عربية `RTL`
- صفحة ترحيب وافتتاحية بصرية وصوتية
- دعم من فريقين إلى 4 فرق
- 10 نقاط لكل إجابة صحيحة
- مساعد ذكي وتلميحات وقراءة صوتية
- نتائج نهائية وشهادة فوز
- مناسبة للمشاركة عبر Teams أو النشر على الويب

## التخصيص السريع

يمكن تعديل الأسئلة من داخل التطبيق أو من ملف `script.js`.

كل سؤال يتكون من:

- `category`
- `prompt`
- `options`
- `correctIndex`
- `explanation`

## الملفات

- `index.html` الواجهة الرئيسية
- `styles.css` التنسيق
- `script.js` منطق اللعبة والأسئلة
- `school-logo.svg` الشعار الافتراضي
