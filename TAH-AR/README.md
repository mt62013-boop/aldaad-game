# TAH-AR | لعبة الفصاحة العربية

لعبة مسابقات تعليمية جماعية تفاعلية في اللغة العربية (بلاغة، فهم، قواعد، تذوق).

- جميع الحقوق محفوظة © طه كامل 2026
- يمكن تشغيل اللعبة محليًا أو رفعها على GitHub Pages.

## الملفات الأساسية
- index.html: صفحة البداية واختيار الدور
- teacher.html: لوحة تحكم المعلم
- student.html: واجهة الطالب
- style.css: التنسيقات العامة
- app.js, teacher.js, student.js: منطق اللعبة
- questions.json: مثال قاعدة بيانات الأسئلة

## طريقة التشغيل
1. افتح index.html في متصفح حديث (يدعم العربية وRTL)
2. اختر دورك (معلم أو طالب)
3. استمتع باللعب!
